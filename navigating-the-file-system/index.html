Navigating the file system

How to move between folders using the terminal

A computer file system is navigable via the command line interface. Understanding how to find your current location and move between system folders is an important first step towards command line proficiency.

Where am I?

The command line interface gives a user access to the file system tree, just like GUIs like Windows Explorer and Finder. It is important to be able to quickly understand where you are within the computer file system when working with the command line, as this will impact the tasks you are able to carry out. If you launch a new terminal session you should usually be able to establish where you are from the command prompt. Where this is ambiguous, you can try using the "print working directory" command. The output from this command will then be printed out in the terminal window below where the command was executed, and the command prompt will reappear beneath that output. For example: 

$ pwd
/Users/sp20532
$
This command prints the absolute path, which can be thought of as the full address to your current location. Relative paths can be used to describe the address of a file relative to your current location. There are short hand notations to help navigate and spell-out path addresses:
~ Your home folder.
. Current working directory.
.. Current working directory's parent, one folder "back" from the current location.
- The directory most recently navigated to before entering the current working directory.
/ The root folder containing the entire file system.
There are a couple of essential commands that are needed to explore and move around the file system:
ls This command lists the content within a specified directory. Simply typing ls will list the contents of the current working directory, while ls .. will list the contents of the directory one folder above the current working directory in the file system tree.
cd This command is used to change directory. To navigate one directory up in the file system tree one could type cd .., and cd ~ to navigate to the home directory.
