Starting a terminal session

How to launch a terminal session and access the command line interface

All operating systems offer a terminal application. This introduction focuses on commands for a Unix-style shell. For Windows users this will require installing a bash shell.

  • Get Started

How you launch the command line interface will depend on which operating system (OS) you are using.

OS

How to access a bash terminal
OS X In Finder go to Applications -> Utilities -> Terminal, or search Terminal in Launchpad or Spotlight
Linux The keyboard shortcut Ctrl+Alt+T is the quickest way to open a terminal when using Linux.
Windows Most Windows systems have Command Prompt as a native command line interface. Newer distributions also have PowerShell. Since this material focuses on Unix-like shells rather than DOS-like shells, these applications are not covered here. Instead Windows users should install a bash shell. Newer Windows distributions have a terminal offered with Windows Subsystem for Linux. Otherwise, Git Bash provides a bash shell.
