Starting a terminal session

How to launch a terminal session and access the command line interface

Command line interfaces, or shells, provide a terminal or console into which a user can issue instructions and provide data to a computer operating system interactively. The level of precision afforded by this method of user input means that tasks can be made reproducible, scalable and can be documented without the ambiguity sometimes introduced when navigating Graphical User Interfaces (GUIs).
